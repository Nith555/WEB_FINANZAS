# Jóvenes con Finanzas

**Jóvenes con Finanzas** es un proyecto web personal desarrollado para aprender y practicar HTML, CSS y conceptos básicos de estructura web y posicionamiento SEO.

El sitio ofrece contenidos sobre educación financiera para jóvenes, con secciones dedicadas al ahorro, la inversión, los créditos, la gestión de deudas y curiosidades del mundo financiero.  
Todos los textos fueron generados con ayuda de herramientas de inteligencia artificial (IA), y el propósito del proyecto es exclusivamente educativo y personal, sin fines comerciales.

---

## Objetivo del proyecto

El objetivo principal ha sido aprender mediante la práctica:
- Mejorar en el diseño y la maquetación web con HTML y CSS.  
- Comprender la estructura de un sitio web con distintas secciones y artículos.  
- Explorar la generación y organización de contenidos con apoyo de IA.  
- Familiarizarse con la creación y uso de sitemaps XML para SEO.

---

## Estructura de la web

El sitio se organiza por secciones temáticas, cada una con varios artículos individuales.

### Página principal
- `https://jovenesconfinanzas.com/`  
  Página de inicio que introduce el proyecto y enlaza a las diferentes secciones.

### Secciones principales
- **Presupuesto y Ahorro**: consejos sobre cómo ahorrar y gestionar los gastos personales.  
- **Inversión para Jóvenes**: introducción a conceptos básicos de inversión, bolsa y criptomonedas.  
- **Gestión de Deudas**: pautas para evitar o controlar deudas de manera responsable.  
- **Créditos y Préstamos**: explicaciones sobre préstamos personales y diferentes tipos de crédito.  
- **Curiosidades Financieras**: artículos divulgativos sobre temas económicos y financieros.

Cada sección contiene varios artículos, como:
- Cómo hacer un presupuesto para estudiantes universitarios  
- Cómo invertir 100 euros siendo principiante  
- Errores comunes al pedir un préstamo joven  
- 7 curiosidades sobre la inflación
